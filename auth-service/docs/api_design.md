# API Design 
- [POST] `v1/auths/login`
  - Request 
    - Body:
      - username|email
      - password
  - Response
    - access_token
    - refresh_token  
- [POST] `v1/auths/refresh_token`
  - refresh_token
- [POST] `v1/logout`
  - refresh_token
- [POST] `v1/register`
  - Request
    - Body:
      - email 
      - username
      - password
      - mobile_number
